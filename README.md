# 📡 RF Spectrum Control System
## 📖 О проекте

**RF Spectrum Control System** — это распределенная система для централизованного управления анализаторами радиочастотного спектра. Система позволяет:

- 🔍 **Удаленно управлять** анализаторами спектра через MQTT протокол
- 📊 **Визуализировать спектр** в реальном времени с помощью интерактивного веб-интерфейса
- 🚨 **Автоматически обнаруживать** сигналы и генерировать тревоги при превышении порогов
- 💾 **Сохранять историю** измерений в базе данных для последующего анализа
- 🔌 **Подключать реальные SDR устройства** (RTL-SDR) или использовать эмулятор для тестирования

### 🎯 Цель работы

Разработка централизованной системы управления анализаторами спектра с возможностью удаленного мониторинга радиочастотной обстановки и автоматического обнаружения сигналов.

## ✨ Основные возможности

### 📡 Управление анализаторами
- Отправка команд на сканирование спектра
- Настройка параметров (RBW, аттенюатор, частота)
- Поддержка множества устройств одновременно
- Автоматическое обнаружение новых устройств

### 📊 Визуализация данных
- Интерактивный график спектра мощности
- Отображение пиков сигналов
- Настраиваемое сглаживание
- Масштабирование и панорамирование графика

### 🚨 Мониторинг и оповещения
- Автоматическое обнаружение сигналов выше порога
- Генерация тревог при превышении уровней
- История тревог с возможностью разрешения
- Цветовая индикация уровня сигнала

### 💾 Хранение данных
- SQLite база данных для хранения измерений
- История всех сканирований
- Экспорт данных в CSV формат
- Статистика по измерениям

## 🛠 Технологии

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| **Backend** | FastAPI / Python | REST API сервер |
| **Веб-интерфейс** | HTML5/CSS3/JavaScript | Пользовательский интерфейс |
| **Визуализация** | Chart.js | Графики спектра |
| **Протокол** | MQTT | Коммуникация с устройствами |
| **Брокер** | Mosquitto | MQTT брокер |
| **SDR библиотека** | pyrtlsdr | Работа с RTL-SDR |
| **БД** | SQLite | Хранение данных |
| **Асинхронность** | Uvicorn | ASGI сервер |
