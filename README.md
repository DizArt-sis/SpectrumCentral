# 📡 RF Spectrum Control System
## 📖 О проекте

**RF Spectrum Control System** — это распределенная система для централизованного управления анализаторами радиочастотного спектра. Система позволяет:

- 🔍 **Удаленно управлять** анализаторами спектра через MQTT протокол
- 📊 **Визуализировать спектр** в реальном времени с помощью интерактивного веб-интерфейса
- 🚨 **Автоматически обнаруживать** сигналы и генерировать тревоги при превышении порогов
- 💾 **Сохранять историю** измерений в базе данных для последующего анализа
- 🔌 **Подключать реальные SDR устройства** (RTL-SDR) или использовать эмулятор для тестирования

### 🎯 Цель работы

Разработка централизованной системы управления анализаторами спектра с возможностью удаленного мониторинга радиочастотной обстановки и автоматического обнаружения сигналов.
